Web Design Essentials

Everything I need to remember when designing websites

Colors & Typography

  • Use high contrast for readability
  • Pick max 2–3 fonts
  • Font sizes: 16px base, 2x for headings

Layout Principles

  • Use a grid system
  • Whitespace improves focus
  • Design mobile-first

UX Basics

  • Clear hierarchy with headings
  • Intuitive navigation
  • Strong call-to-actions
